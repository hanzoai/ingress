# Migration: Steps needed between the v2 versions

The multiple minor versions of Hanzo Ingress v2 introduced a number of changes,
which may require one to update their configuration when they migrate.

For more information about the changes between Hanzo Ingress v2 minor versions, please refer to the [v2 documentation](https://github.com/hanzoai/ingress/blob/main/docs/content/v2.11/migration/v2/).
